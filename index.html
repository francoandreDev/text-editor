El uso recomendado es para la creación y la edición de archivos de texto. Si el archivo ha sido editado externamente ciérrelo y use el botón abrir antes de seguir trabajando. En windows tras dos segundos de inactividad se guardará el archivo y para la primera vez pedirá confirmación. En android tras 15 segundos de inactividad se descargará el archivo, si el nombre de archivo ya existe pedirá confirmación.
